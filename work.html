movie_api

movie_api is a server-side component of a movies web application. It provides users with access to information about different movies, directors and genres. Users are able to sign up, update their personal info and create a list of their favorite movies.

the-myFlix-client

myFlix is built using MERN (MongoDB, Express, React & Node.js) and is a single-page application. It builds further on its existing server-side code (the aforementioned movie_api). Users are able to register an account, log into their account, filter & sort movies, view movie details, add movies to their favourites list and much more!

meet

meet is a serverless, progressive web application (PWA) built with the React framework. It uses a test-driven development technique. The application uses the Google Calendar API to fetch upcoming events. Users can filter events by city, show/hide events details, specify number of events, add an app shortcut to their home screen and even use it offline!
